# Requisitos para iniciar el proyecto

- Docker
- Docker Compose
- TypeScript
- Node.js

# Pasos para iniciar el proyecto

- Ejecutar `yarn install`
- Ejecutar `docker-compose up -d`
- Ejecutar `docker ps` para verificar que los contenedores esten corriendo
- Ejecutar `yarn start`
